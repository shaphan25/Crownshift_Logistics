# **App Name**: Crownshift Logistics

## Core Features:

- Service Overview: Display a comprehensive list of services offered by Crownshift Logistics.
- Instant Quote Generator: Allow users to enter the origin, destination, and dimensions of their package to receive an instant quote powered by AI.
- Real-Time Tracking: Provide real-time tracking of shipments, allowing users to monitor the status and location of their packages.
- Contact Form: Enable users to easily submit inquiries or requests through a contact form.
- Testimonials: Showcase positive testimonials from satisfied customers to build trust and credibility.
- CMS Integration: Allow company administrators to upload images.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent trust, reliability, and logistics.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Yellow-Orange (#FFB300) to highlight important interactive elements such as calls to action, quote button.
- Headline font: 'Poppins' sans-serif font for a contemporary and fashionable look.
- Body font: 'PT Sans' humanist sans-serif font for a balance of readability and a modern look.
- Use clear, modern icons to represent various logistics services, such as shipping, tracking, and delivery.
- Design a clean and structured layout with clear sections for each service, tracking information, and contact details.
- Use subtle animations to enhance user engagement and provide feedback, such as loading animations, transitions, and hover effects.
- Use the company logo (image provided) in the header and footer of the website.
